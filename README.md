# Atlas Web App

## Running the app

To run:

```
npm run install
npm run build
npm start
```

## Development

When developing, its useful to run build in watch mode.

In one terminal tab:

```
npm run watch
```

In another terminal tab

```
npm start
```

## Deploying the app

To deploy

```
npm run install
npm run build
```

Then copy everything in the /public directory to your server
